Welcome to PowerUI!

The HTML for this example is in a file called Basics.html sitting alongside this scene in your project.

To add a UI to a new scene..
Go to GameObject > Create Other > Simple Main UI. This creates a gameobject which you can attach a .HTML file to.

To quickly make a .HTML file..
Right click in your project and go to PowerUI > New HTML File.

Need help? No problem!
PowerUI can do a vast amount of stuff so make sure you check out our help guides or if you're stuck please do get in touch!

For the latest updates and free extras see My PowerUI.
